## Week 1: “Hello World!” — How to Talk to Robots

**Objective:** Get comfy with C++ and Python, learn to tame the electrons, and impress robots enough to make them obey your commands.

### Resources

1. **C++**
    - [Learn C++ (It’s not just for wizards)](https://www.learncpp.com/)
    - **Recommended Reading:** Start with Chapter 1 (Basics). Stop at Chapter 3 once you've mastered making loops dance.

2. **Python:**
    - [Python Crash Course by Eric Matthes](https://khwarizmi.org/wp-content/uploads/2021/04/Eric_Matthes_Python_Crash_Course_A_Hands.pdf)
    - **Suggested Reading:** Skim through Chapters 1–3 and get cosy with variables, loops, and indentation.
    - [W3Schools Python Tutorial](https://www.w3schools.com/python/)

3. **Introduction to Electronics:**
    - Brush up on voltage, current, and resistance—the holy trinity of electronic sorcery. Try SparkFun’s Handy Tutorials for an easy-to-digest start.

### Assignments

1. **C++:**
    - Write a program that performs basic arithmetic (addition, subtraction, multiplication, division).
    - Implement a “Guess the Number” game: the computer picks a random number, and you guess it.

2. **Python:**
    - Build a script that calculates the factorial of a number using recursion.
    - Solve Level 1 questions from the [Python Challenging Programming Exercises Repo](https://github.com/zhiwehu/Python-programming-exercises).

## Week 2: Git, GitHub, and Terminal

**Objective:** Master the essentials of version control with Git and GitHub, and become comfortable navigating through the Linux terminal.

### Resources

1. **Introduction to Git and GitHub:**
    - [Git and GitHub Crash Course - Video](https://www.youtube.com/watch?v=tRZGeaHPoaw)

2. **Basic Terminal Commands:**
    - Master common commands: `cd`, `ls`, `pwd`, `mkdir`, `rm`, `cp`, `mv`, `nano`.
    - [Linux Terminal Basics Guide](https://homepages.uc.edu/~thomam/Intro_Unix_Text/File_System.html)

3. **Git Basics:**
    - [Git Basics Article](https://medium.com/@tom.puntillo/what-is-git-c150dee640a)
    - [Git Documentation (Pro Git Book)](https://git-scm.com/book/en/v2)

4. **Linux Fundamentals:**
    - [Linux Crash Course Playlist](https://www.youtube.com/playlist?list=PLT98CRl2KxKHKd_tH3ssq0HPrThx2hESW)

### Assignments

- **Linux Challenge:** Complete the [OverTheWire Bandit](https://overthewire.org/wargames/bandit/) game up to Level 15 to master the CLI.
